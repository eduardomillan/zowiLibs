Arduino Libraries for Zowi

This document describes every Arduino library in the Zowi ecosystem and how to use them in your own sketches.

Installation

Copy each library folder into your Arduino libraries directory:

  • Windows: My Documents\Arduino\libraries\
  • macOS: Documents/Arduino/libraries/
  • Linux: libraries/ inside your sketchbook folder

After copying, restart the Arduino IDE. You should see them under Sketch → Include Library.

1. Zowi — Main Robot Library

This is the top-level library. It depends on Oscillator, LedMatrix, US, BatReader, Servo, and EEPROM.

For the complete Zowi class API (every method, parameter, default, and return type), see the Zowi API Reference.

Initialization

#include <Zowi.h>

Zowi zowi;

void setup() {
    zowi.init(2, 3, 4, 5);
}

Motion

zowi.walk(4, 1000, FORWARD);
zowi.turn(4, 2000, LEFT);
zowi.jump(1, 2000);
zowi.bend(1, 1400, LEFT);
zowi.shakeLeg(1, 2000, RIGHT);
zowi.updown(1, 1000, 20);
zowi.swing(1, 1000, 20);
zowi.tiptoeSwing(1, 900, 20);
zowi.jitter(1, 500, 20);
zowi.ascendingTurn(1, 900, 20);
zowi.moonwalker(1, 900, 20, LEFT);
zowi.crusaito(1, 900, 20, FORWARD);
zowi.flapping(1, 1000, 20, FORWARD);

Sensors & Battery

float dist = zowi.getDistance();
int noise = zowi.getNoise();

LED Mouth

Sounds & Gestures

zowi.playGesture(ZowiHappy);

2. Oscillator — Servo Motion Engine

Generates smooth sinusoidal motion for each servo. Zowi uses 4 instances internally (one per leg joint).

#include <Oscillator.h>

Oscillator osc;

void setup() {
    osc.attach(9);
    osc.SetA(45);
    osc.SetO(90);
    osc.SetT(2000);
    osc.SetPh(0);
}

void loop() {
    osc.refresh();
}

3. LedMatrix — LED Mouth Display

Controls a 5x6 LED matrix via three 74HC595 shift registers (data=11, clock=13, latch=12 by default).

LedMatrix matrix;
matrix.writeFull(0b000001000010000100001000001111);
matrix.setLed(1, 1);
matrix.unsetLed(3, 4);
matrix.clearMatrix();
matrix.setEntireMatrix();

4. US — Ultrasonic Distance Sensor

US us;
us.init(8, 9);
float cm = us.read();

5. BatReader — Battery Monitor

BatReader bat;
double volts = bat.readBatVoltage();
double percent = bat.readBatPercent();

6. ZowiSerialCommand — Serial Command Parser

ZowiSerialCommand sc;

void handleMove() {
    char *arg = sc.next();
}

void setup() {
    Serial.begin(9600);
    sc.addCommand("MOVE", handleMove);
    sc.addDefaultHandler(handleUnknown);
}

void loop() {
    sc.readSerial();
}

7. EnableInterrupt — Pin Interrupts

#include <EnableInterrupt.h>

void buttonPressed() { }

void setup() {
    pinMode(7, INPUT_PULLUP);
    enableInterrupt(7, buttonPressed, FALLING);
}

Typical Sketch Structure

#include <Zowi.h>
#include <ZowiSerialCommand.h>

Zowi zowi;
ZowiSerialCommand sc;

void handleWalk() {
    char *arg = sc.next();
    int steps = arg ? atoi(arg) : 4;
    zowi.walk(steps);
}

void setup() {
    Serial.begin(9600);
    zowi.init(2, 3, 4, 5);
    sc.addCommand("WALK", handleWalk);
}

void loop() {
    sc.readSerial();
    if (zowi.getDistance() < 15) {
        zowi.turn(4, 1000, LEFT);
    }
}
