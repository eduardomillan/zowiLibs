Initialization

    int YL,
    int YR,
    int RL,
    int RR,
    bool load_calibration = true,
    int NoiseSensor      = PIN_NoiseSensor,
    int Buzzer           = PIN_Buzzer,
    int USTrigger        = PIN_Trigger,
    int USEcho           = PIN_Echo
);
ParameterTypeDefaultDescription
YLint—Servo pin: left hip
YRint—Servo pin: right hip
RLint—Servo pin: left foot
RRint—Servo pin: right foot
load_calibrationbooltrueLoad trim values from EEPROM[0..3]
NoiseSensorintA6Analog pin for noise sensor
Buzzerint10Digital pin for buzzer
USTriggerint8Ultrasonic sensor trigger pin

Attach & Detach

attachServos()

Attaches all four servos. Called automatically by init() and _moveServos().

detachServos()

Detaches all four servos, freeing them. Called automatically by home().

Calibration Trims

setTrims(YL, YR, RL, RR)

ParameterTypeDescription
YLintTrim offset for left hip
YRintTrim offset for right hip
RLintTrim offset for left foot
RRintTrim offset for right foot

saveTrimsOnEEPROM()

Saves current trims to EEPROM[0..3]. Loaded automatically on boot if load_calibration=true.

Home & Rest State

home()

Moves all servos to 90° over 500 ms, then detaches them. Sets rest state to true. No-op if already resting.

getRestState()

Returns true if Zowi is in rest position.

setRestState(state)

Manually overrides the rest state flag.

Sensors

getDistance()

Returns distance in cm from the ultrasonic sensor. Returns 999 if no echo.

getNoise()

Returns averaged noise level from the analog noise sensor (pin A6). Averages 2 readings with 4 ms delay.

Battery

getBatteryLevel()

Returns battery percentage (0–100%). Averages 10 readings. First read discarded.

getBatteryVoltage()

Returns battery voltage (3.25V–4.2V). Averages 10 readings. First read discarded.

ParameterTypeDescription

clearMouth()

Clears all LEDs on the mouth matrix.

Plays a predefined song (0–18). See Song IDs.

Gestures

playGesture(gesture)

Executes a complex gesture combining movement, sounds, and mouth animations. Returns Zowi to home with happyOpen when finished.

Gesture IDs: 0–12. See Gesture IDs.
